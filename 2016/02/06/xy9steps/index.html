严新九步冥想方法

原文出自: http://www.daode99.com/view.php?tid=134&p=NDM0MTY=

引文：
我相信冥想对身体的调节和增强身子和环境沟通的能力。这是小时候曾经修习过的严新老师的九步方法。
我不喜欢把这个方法成为气功方法，用大众了解下的气功概念来概括这种冥想方法实在太浅薄。

该文是有人花时间将严新老师的报告录音一个字不差纪录下来然后配图的，做得很好。但我个人理解，觉得如果严老师希望他的修炼方法真的得以推广从而可以帮助人，必须和气功做一个了断。我因此而将所有气功字眼全去掉。希望严老师不要见怪，怪的话就来找我也行。

开功

现在简要介绍一种方法。坐在凳子上的同志，请坐在凳子的三分之一至四分之一处，不要靠在靠背上，能够盘腿的就盘腿。一般来说盘腿练功，比坐在凳子上效果好。坐着练功比站着练功安全一些，躺着练功更安全。站着的两腿分开一点，比肩宽一点，膝盖微微弯曲一点，不要站得太直了。


第二点，请大家腰要伸直（要微微地收缩小腹腰才能够标准的伸直，实际上这不叫伸腰，而是叫塌腰）。

第三点，要注意头正颈直，下颌微微内收。请将头微微的低一点，勾一点，下颌内收一点，请将头向下、向前低到1到3公分左右，这样才能头正、颈直，下颌内收。

第四点，请大家双手放在合适的位置，垂肩沉肘、腋下悬空，两手重叠，掌心向上，男左手在上（女相反），放在小腹处。两手大拇指尖轻轻挨着，注意五指微微分开，小指伸直，向手背方向翅一点，因为小指头与心包经有紧密地连联系，所以要调心，这一点很重要。身体好的、希望反映大的同志，两手掌之间可以离开3至7公分，大拇指尖可以不接触。

第五点，牙齿要微微的分开点，不要咬在一起，舌尖微微地上翅，顶在上门牙根里面；有心脏病的同志，舌尖应该轻轻地顶在下门的牙根里面。有神经衰弱、神经官能症和以前有过精神分裂症病史以及情绪很不稳定、心情烦躁的同志、有很心里包袱的这一部分同志，要将舌头向下后弯曲，舌尖轻轻地顶着舌根的系带上。血压正常而身体肥胖需要下降体重的这部分同志，可以将舌头平放在口腔中间，不要挨着口腔上下腭。但血压低的同志不能这样做。以上舌头大的四种姿势的调节，每一个同志只需要选择一种就行了。

第六点，请大家眼睛放松放低，向下看着你自己的鼻尖，减少光线的刺激，容易入静。希望出现大反映或者身体比较好的同志，可以闭上眼睛，闭上眼睛容易出现大的反映，这样也更容易入静。


这是六个方面的开功姿势的调整、调节，仅仅作为开始练功的姿势。

凡要学练我所讲的修功方法的同志，都要注意收功的方法，我所讲过的方法一定要收功，下面我讲收功的方法。我讲的收功方法大同小异，收功方法，无论是讲的那一种练功方法，都可以采用同样一种、同样一些收功方法去收功。

第一步

第一步：在松静自然的功态基础上，想一下小腹里边有一朵正在开放的荷花。要想着那朵荷花中有赤，橙，黄，绿，青，蓝，紫等颜色，以及一些光线。想着有多种颜色多种光线由小腹里边的荷花中发出来，把五脏六腑都照着了。
这个意念活动要保持一段时间，尽量地在练功活动中保持有这么一个意念。要始终似想非想，好象是小腹里边出来了一朵荷花，又象是没想出来，没想出来也没关系，它是似想非想的。荷花的光线将五脏六腑逐渐逐渐照得明显，有人就可能在这个过程中把自已的五脏六腑都看清楚了。

如果看到了五脏六腑不要紧张，不要害怕，有可能是意念上想的，也有可能是真的看到了。接着进一步调整意念活动，想着荷花在不断地随着呼吸的进行在开放，在收缩。

在这个过程中，注意数自已的呼吸次数，有多大年龄就数多少次。在数呼吸次数的时候，一呼一吸算两次，即一呼一吸各算一次。

数够自已年龄大小的呼吸次数以后，又重复从 1 开始。可以重复自已年龄大小的呼吸次数七遍至四九遍作为练第一步的最短时间。

在数呼吸次数的时候，观察荷花随着吸气在收缩，随着呼气，荷花在开放，同时进一步调整意念活动。在吸气的过程中，再想一下全身毛孔打开了，刚才所说的重要穴位，重要部位，好象也在吸气时打开了一样，呼气的时候全身毛孔和重要穴位好象关闭了似的。

如果今后练功身体不舒服，要考虑先治病的话，就要先注意在呼气的时候是毛孔打开的。这里讲的重在注意强身，因此先要注意意念活动调节在吸气的时候是全身毛孔和重要的穴位打开，在呼气的时候相反关闭了。

数够自已年龄大小的七遍到四十九遍呼吸次数以后，就可以逐渐进入第二步。

第二步

如单独练第二步，可以从开始做。也就是将第一步功收功以后再做第二步，也可以不收功就接着调整意念活动做第二步功。

将意念活动调整到下面一些内容上：主要是吸气的同时要想，小腹里面荷花的光线将心脏照得特别明显，同时想到心脏在吸气过程中，也在变化。

在呼气的时候，既要想又要观察小腹里边的荷花在开放，心脏在收缩，皮肤毛孔也在收缩 ; 在吸气的时候想着自已体内的荷花在收缩，心脏在舒张，全身的毛孔和重要穴位在开放。

在数自已呼吸的过程中，重点要用意念观察里边的变化，如果意念中能够将自已的五脏六腑照得比较清楚为好。

就这样意念调整，练到有反应或者说自已觉得感觉还比较明显以后，就可能进入收功状态，也可以接着练第三步。

第三步

如果是单独练第三步，也可以从开功开始。不收功接练第三步功也行。

第三步：将意念活动再进一步调整。开功后，要逐渐加强意念，想自已父母亲年轻，健康的样子。一般说来是男性练功先想母亲年轻健康的样子 ; 女性练功先想父亲年轻健康的样子。并不一定要想出来，要似想非想。将自已母亲或者父亲似想非想地想出一个模糊的图像，再慢慢的将他们想到自已当面来，想到自已面前七步之内。

然后进一步想，母亲或者父亲的这个影子跟自已的动作基本上是一样的在练功，就象跟他自已在对练一样。然后就进一步数呼吸次数，要数自已年龄加上母亲年龄，自已年龄加上父亲年龄之后和的呼吸次数。也就是说现在练功的时间要延长了。无论是有的人父母亲去世了或者还在，都是以算到练功时的年龄为准。

在数呼吸次数过程中，要注意随着自已呼气或吸气时观察自已体内和自已意念活动中的母亲或父亲影子体内的双重荷花，心脏，皮肤毛孔和重要穴位的变化。这个意念活动很不好调整，在呼气的时候既要想又要观察小腹里面的荷花在开放，心脏在收缩。皮肤毛孔也在收缩，同时观察对面想的那个亲人的影子也是这样在变化 ; 在吸气的时候想着自已体内和对面影子的体内的荷花在收缩，心脏在舒张，全身的毛孔和重要穴位在开放。这个意念活动需要高度的调整，才能调整到比较标准的状态。注意体会自已的体内变化。

这个时候很容易在练功有基础的情况下，感受到自已意念活动中母亲或者父亲的身体不舒服的情况。练这种很容易为亲人治病，常常是一人练功全家受益。

小孩练功容易为父母亲治病。如果在这个时候出现了一些感觉，特别是不舒服的感觉，常常意味着母亲或者父亲可能某个地方有类似不舒服的反应，也许在过去，现在，或者不久的将来，可能出现类似的反应。将这种意念活动调节到好象看到了，想到了就真能看到似的。

在这个过程中，不要轻易把自已感到难受的地方当作是自已生病了。这种难受过程是由于有可能在为自已的亲人调节治疗某些病症，所以在这个时候如出现一些疼痛，发涨或者难受的情况，常常对亲人有相当大的调节治疗作用。

第三步练到自已年龄加上意念活动中男性的母亲，女性的父亲年龄之和的七到四十九遍后，就可以进入收功状态，进行收功。如果有时间也可以延长，或者说连续练第四步。

第四步

在开功或者入静有感觉以后，又要进一步调整意念活动，将第三步功中用意念想的那个影子加意念调整到自已身上来，将刚才要大家想的母亲或者父亲健康的影子慢慢地想到自已身上来。

男性将自已母亲的影子，女性将自已父亲的影子想到自已身上来，好象就在自已身上一样。将这个意念调节以后，重复数自已年龄加上自已母亲或者父亲年龄大小的呼吸次数的呼吸意念活动。

在吸气的时候，要用意念想或用内视的办法来观察自已小腹里面的荷花有双重影子。双重荷花的影子在收缩 ; 心脏也是双重心脏。双重影子心脏在舒张，全身的皮肤毛孔和前面提到的重要穴位也是双重影子似的在舒张，在开放，在向体内进光，进气，进雾状的物质，或者进其它图像。

在呼气的时候就刚好相反，想着自已小腹里的双重影子在收缩，皮肤和重要穴位也是双重影子在关闭，这是这一步需要调整的意念活动。练这一步治疗效益就有可能比上一步强一些。

由于练这一步偏重于进一步追踪治疗和加强治疗，所以它有可能在反应上有一些另外的变化。这两步常常需要连起来练习，练一段时间如果感觉比较明显，就可以练下一步。

第五步

第五步：在开功或入静基础上调整意念活动。现在想自已刚才没有想到的父母亲中另一位亲人的影子。男性想父亲，女性想母亲年轻健康的样子或者影子。要把他想到自已面前七步之内，与自已的动作一样，在与自已对练功。第五步刚好与第三步相对应。

如果似想非想的好象想出来了这么一个影子，那么就进一步的调整意念活动。这个意念活动的调整与第三步基本上是一致的，同样是在练功过程中注意呼吸，意念二者结合，也就是说同样要数自已呼吸的次数并同时观察体内有关意念图像等方面的变化。现在呼吸次数是自已年龄大小加上父母亲年龄的总和 ; 也就是说把三个人的年龄大小之和作为练功时间长短的基数。此时练功的时间也就延长了。

在数自已吸气的同时要观察自已体内双重荷花影子在收缩，还要同时观察与自已姿势一样的父亲或母亲体内的荷花也在同时收缩，双边观察。这一步比第三步更难一些。

在荷花收缩的同时，观察自已体内心脏和附在自已身上影子的心脏以及面前的那个亲人体内的影子的心脏在舒张，自已以及自已身上影子的全身皮肤毛孔和重要穴位以及对面意念中亲人影子的皮肤毛孔和重要穴位在吸气。

荷花收缩，心脏舒张的同时毛孔和重要穴位在开放，有多种颜色的光线，雾状的东西或者其它的图像在进入自已体内，自已身上影子的体内以及意念活动中亲人影子的体内，这就是三重意念活动合在一起调节。

此意念很不容易调整到标准状况，调整不到标准状况没关系，只要加这个意念活动它就可以自动调节。在数呼吸次数的过程中，有可能感受到父亲或者母亲的一些不舒服情况。

男性练这种，容易感受到父亲的一些病症，女性练这种，就容易感受到母亲的一些不舒服和病症。但也有可能将父母亲两个人的不舒服的情况都感应到自已身上来，在这个过程中，出现的反应不要当作自已体内的病态。

练到自已年龄加上父母亲年龄之和呼吸次数的七遍到四十九遍以后，就可以收功。也可以不收功继续练第六步。

第六步

第六步与第四步基本一致，就是男性把自已父亲，女性把自已母亲的影子想到自已身上来，好象这个影子到自已身上了。

当似想非想的将这个影子想到自已身上，就需要数自已年龄大小加上父母亲年龄大小总和的呼吸次数七遍到四十九遍。在数呼吸次数的同时，用意念观察自已体内荷花心脏和皮肤毛孔以及重要穴位的变化，同时又要观察自已身上还有母亲或者父亲合起来的两个影子的荷花，心脏，皮肤毛孔和穴位的变化情况，也就是要将荷花从意念上想成三个影子，将心脏，皮肤毛孔和重要穴位同样想成三个影子。

在数呼吸过程的同时，要想出自已小腹内荷花有三重影子在变化，在呼气的时候，荷花有三个影子在开放，心脏也有三个影子在收缩，皮肤毛病孔以及重要穴位也有三个影子在关闭 ; 在吸气的时候就要想小腹里面的荷花有三重影子在收缩，心脏有三重影子在舒张，皮肤毛孔和重要穴位也有三重影子在开放，在进不同颜色的光线，不同的雾状物质或者图像。就这样始终注意深，长，细，匀的呼吸，并注意观察。

这个过程容易为自已父亲或者母亲进一步治疗某些病症，或者说为自已父母亲同时治疗某些病症。

由于自已与父母亲，兄弟，姐妹或者爷爷，奶奶等有一些遗传关系，这就有可能为全家人调节治疗某些病症。

因此，在这一步中，同样有一些不舒服的情况。不舒服也是正常的，不要误认为是病症。这步练到一定时间，有收益以后，可以练下一步共法。

每次练的时候，需要练够七至四十九遍，作为一次练功时间长短的最低标准，就可以进入收功。收功以后，可以干其它事情，也可以再练下一步，也可以不收功接着练下一步。

第七步

第七步要在第六步的基础上再进一步调节，要将前面的几步中意念中的父母亲的两个影子合为一个影子，再把自已的身体合在父母亲的影子中，就是三个影子合为一个影子。

进一步想这个影子在自已小腹里边荷花 上站着，这个意念尽量地想出来，想出来以后再重复前面第六步的意念，呼吸活动。也就是注意数自已加上父母亲年龄大小的呼吸次数，在数呼吸过程中，同样要观察自已小腹里边荷花 上站的影子，以及心脏，皮肤毛孔，重要穴位的变化情况。

吸气时想小腹里面的荷花 在收缩，同时想站在荷花 上面三个影子合为一个影子的小腹里面的荷花 也在收缩，心脏在舒张，皮肤毛孔和重要穴位在开放 ; 在呼气的时候要想自已小腹里的荷花 以及三个影子合为一个影子的小腹里的荷花 在舒张，心脏在收缩，皮肤毛孔和重要穴位在关闭，数够自已年龄加上父母亲年龄大小的呼吸次数的七至四十九遍作为一个基数，作为练功时间长短的一个标志。

这一步就更可能为自已和自已父母亲以及其他的亲人调节治疗病症。

在这个过程中的所有反应也都是正常的不要把它当作异常对待，练够一定的时间又可以进行收功。收功以后可以从事其它的活动，也可以再进入下一步练功。

第八步

第八步的意念活动是只注意自已的呼气和吸气中的任何一种，在这里最好是先注意吸气，不注意呼气，也就是说第八步主要是数自已吸气不要数呼气，只数自已年龄大小的吸气次数，数够自已年龄大小次数的七至四十九遍吸气次数，作为练功时间长短的标记。

在数自已吸气的同时，想荷花 在收缩 ; 同时要想站在荷花 上面的那个影子在往荷花 里面进入，慢慢地进入荷花 里边。每数一次呼吸，荷花 上面的影子就往荷花 里面进入一部分，数够自已年龄大小的呼吸次数以后，这个影子就进入荷花 里边了。

在数吸气的过程中，注意荷花 始终在收缩，心脏在舒张，皮肤毛孔和重要穴位在开放 ; 呼气的时候，不要去注意荷花 ，心脏，皮肤毛孔，意念始终想着荷花 在不断的收缩，影子在进入荷花 里边。

数够年龄大小次数的七至四十九遍以后，可以收功。

第九步

第九步 意念活动的调整也是比较简单的，就在第八步的基础上，注意数自已的呼气，不数吸气，数自已年龄大小次数的呼气次数七至四十九遍作为练功时间长短的标准。

在数呼气的过程中，同时用意念观察荷花 在开放，心脏在收缩，全身的皮肤毛孔和重要穴位在关闭。

也就是说在这一步中要进一步的封闭自已，关闭自已，进一步的进行内功内练了。将气汇聚到体内以后，就要进一步深化锻炼了。

数够自已年龄大小七至四十九遍的呼气次数，每数呼气次数一遍，荷花 就开放一部分，荷花 里边的三个图像的影子就长大一岁，当数够你现在这么大的年龄的时候，荷花 里边那三个图像的影子就刚好长到了你自已现在这么大的个子，就是自已的身体，好象就是自已了，这就是第九步 的意念活动。

如果数够七至四十九遍要收功的时候，就需要进一步想，想自已就是站在荷花 上面。练到此时荷花 长得很大，有可能把自已身体包围起来了，也有可能就在自已的脚下。

这个时候，有人可能身体会感觉得很轻，可能会发生其它的一些轻功方面的变化，但一般人不容易做得理想。

收功

收功的时候，眼睛一定要睁开，双手合掌，掌根对着胸部，挨着胸部，指头向前，要用劲，眼睛要睁开，收功的时候不能闭眼睛了，眼睛正开可以看远处。

请注意下面收功的一些意念，先想着双脚用劲，进一步想双脚的十根指头用劲，再进一步想双脚的小指头用劲，再进一步想双脚的大指头用劲，接下来想双大腿用劲，双小用劲，要收缩前后二阴，收缩小腹，收缩腰部，接下来牙关紧闭，上下牙要咬在一起，舌头恢复自然姿势，在口腔里边旋转：男性反时针方向旋转四十九圈，女性顺时针方向旋转三十六圈。有唾液可以在口腔里鼓漱，慢慢地吞下去，不要吐掉。

在气功里面唾液叫金津玉液，不要把它吐掉，慢慢地吞下去，可以起着一定的助消化，或者治疗咽炎或者其它某些病症的作用。接下来双手特别用劲推，全身用推，好象身体在震颤似的，双手用大劲，尽量用劲。牙关紧闭。

接下来双手快速搓掌，搓得发热。这个时候身体可以自由地活动一下，就不要太标准的坐着了。搓完掌以后，手掌对着眼睛，手掌挨着脸，也可以隔开一点，想一下手掌的光线，全身的光线通过手掌掌心进入眼睛，保护和改善视力。接着双手挨着脸往上擦的同时深吸气


擦到头顶正中百会穴，再往下擦，呼气，擦到下颌下边，双手合掌，然后再往上擦。就这样反复做六至七次左右。

男性可作七次，女性可作六次，超过这个数没关系。深呼吸，象打个哈欠一样的深呼吸。擦到感觉比较清醒，手就交叉，两手虎口相交，男性左手在外边，女性右手在外边，把手按在肚脐部位，稍稍用点力。

想一下全身的光线通过手掌进入肚脐，再想一下刚才想的荷花和影子以及所有光线都进入本肚脐，肚脐里边有一个点在闪亮，再想一下有光线围绕肚脐旋转，男性反时针方向旋转 49 圈，女性顺时针方向旋转 36 圈。但次数也可以灵活掌握。手可以动，也可以不动，也可以想一就行了。

最后收功把手拿开，恢复到自然姿势，就算收功了。
